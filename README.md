## 📁 Project Structure
The project is split into three core files for maximum maintainability:

```text
/
├── index.html       # Main HTML structure (Home, About, Skills, Projects, etc.)
├── css/             # Stylesheets
│   └── style.css    # Clean, categorized Vanilla CSS (no @import)
├── js/              # Client-side logic
│   └── script.js    # Interactive effects and navigation
└── assets/          # Media & Documents
    ├── images/      # Profile photos, certificate images, and icons
    └── docs/        # Downloadable assets like CV (PDF)
```

## 🚀 Quick Start & Customization

### 1. **Setup Profile Photo**
- Add your image to `assets/images/profile.jpeg`.
- The code is pre-configured to load this file automatically in the HERO and ABOUT sections.

### 2. **Add Certificates**
- Place certificate JPGs in `assets/images/` as `cert-1.jpg`, `cert-2.jpg`, ..., `cert-6.jpg`.
- This will automatically populate the interactive certificate gallery.

### 3. **Manage Your CV**
- Save your resume/CV as `assets/docs/MyCv.pdf` for the download button.

### 4. **Update Contact Info**
- Replace placeholder emails and phone numbers in `index.html`.
- For the WhatsApp link, update the `wa.me/92XXXXXXXXXX` link with your actual number.

## 🌐 Deployment

### **Netlify (Recommended)**
1. Drag and drop the root folder into **[Netlify Drop](https://app.netlify.com/drop)**.
2. Your site will be live instantly.

### **GitHub Pages**
1. Push this repository to GitHub.
2. Go to **Settings > Pages** and select the `main` branch as your source.
3. Your site will be live at `https://yourusername.github.io/portfolio`.
